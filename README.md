# 🤖 AI Evaluation Project

An end-to-end AI Evaluation Framework built using Python, Large Language Models (LLMs), Prompt Engineering, Git, GitHub, and Streamlit.

---

## 🚀 Project Overview

This project demonstrates how to build, evaluate, and deploy modern LLM applications using industry-standard AI engineering practices.

The application will cover:

- LLM Integration
- Prompt Engineering
- AI Evaluation Metrics
- Streamlit UI
- Git & GitHub
- Deployment
- CI/CD
